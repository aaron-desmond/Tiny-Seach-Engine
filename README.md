# CS50 Tiny Search Engine (TSE)

This project implements a `crawler`, `indexer`, and `querier` to create a tiny search engine using modules in the `common` and `libcs50` libraries. For more information on each module, see the corresponding markdown files.


## Compilation
To compile all of the directories, run:
```bash
make
```

To clean all of the directories, run:
```bash
make clean
```
